# curriculum

Curriculum using CSS Grid , Flexbox and React

## Getting Started

First just

```
git clone https://github.com/joaopedrodcf/curriculum.git
```

Then

```
npm install
```

Finally

```
npm start
```

Now its ready to use

### Prerequisites

You need to install Node.js
